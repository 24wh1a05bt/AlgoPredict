#AlgoPredict – Decentralized Prediction Market

Description:
AlgoPredict is a full-stack decentralized prediction market platform on Algorand. Users can create markets, trade shares in real-time, and claim rewards automatically through smart contracts, ensuring trust, transparency, and security.

Tech Stack:

Smart Contracts: Algorand Python (AlgoKit)
Backend: Node.js (Express) / Python API
Frontend: React (Vite) + Tailwind CSS
Wallet Integration: Pera Wallet
Charts: TradingView Lightweight Charts
Blockchain Data: AlgoNode / Algorand Indexer API

Core Features:

Create & manage prediction markets
Live orderbook for buying/selling shares
Automated outcome resolution & reward distribution
Multi-source oracle simulation
Real-time market data & interactive charts
Wallet connection & secure transaction signing

Setup Instructions:

Clone the repo:

git clone https://github.com/yourusername/AlgoPredict.git

Install backend dependencies:

cd backend
npm install

Install frontend dependencies and run:

cd frontend
npm install
npm run dev
Deploy smart contracts using AlgoKit.
Connect Pera Wallet to interact with markets.

Outcome:
A trustless, decentralized platform for predictions demonstrating full-stack blockchain integration.
